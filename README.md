# 🤖 ОБНУЛЕННЫЙ Crypto Analyzer Bot

Telegram бот для анализа криптовалютных проектов с использованием CoinGecko API.

## 🎯 Возможности

- ✅ Полный анализ любого крипто-проекта
- ✅ Быстрое получение цены и данных
- ✅ Сравнение двух монет
- ✅ Топ 20 монет по объёму торгов
- ✅ Расчет уровня риска
- ✅ Красивое форматирование для постинга

## 📋 Команды

```
/start - Начать работу с ботом
/help - Справка по всем командам
/analyze [монета] - Полный анализ проекта
/price [монета] - Быстрая цена
/compare [coin1] [coin2] - Сравнение двух монет
/top - Топ 20 монет по объёму

/ta [монета] - Технический анализ (SMA/EMA/MACD/Bollinger/RSI)
/exchanges [тикер] [quote] - Цена и объём на Binance/Bybit/Kraken/OKX
/screener - Топ-15 по объёму, отфильтрованные по RSI
/news [монета] - Свежие новостные заголовки

/community - О сообществе и тарифах
/invite - Персональная реферальная ссылка
/rules - Правила сообщества
```

Команды `/ta`, `/exchanges`, `/screener`, `/news` — премиум-аналитика: 2 бесплатных запроса в сутки на пользователя (общий лимит), дальше апсейл на платный тариф через `/community`.

## 🚀 Быстрый старт

### Локальное тестирование (5 минут)

1. **Клонируй репозиторий:**
   ```bash
   git clone https://github.com/[твой-аккаунт]/obnylennyy-crypto-bot.git
   cd obnylennyy-crypto-bot
   ```

2. **Установи зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Создай `.env` файл:**
   ```bash
   cp .env.example .env
   # Отредактируй и добавь свой TELEGRAM_TOKEN
   ```

4. **Запусти бота:**
   ```bash
   python crypto_bot.py
   ```

### Развертывание на Railway (3 минуты)

Смотри **DEPLOYMENT.md** для пошаговой инструкции!

## 🛠 Технический стек

- **Python 3.9+**
- **python-telegram-bot** — Telegram Bot API
- **requests** — HTTP запросы
- **ccxt** — единый доступ к данным Binance/Bybit/Kraken/OKX
- **CoinGecko API** — цены, market cap, история цен для теханализа
- **CryptoCompare News API** — новостные заголовки

## 📊 Данные анализа

Бот анализирует:
- 💰 Текущая цена (USD)
- 📈 Изменения (24h, 7d, 30d)
- 💎 Market Capitalization
- 📊 Trading Volume (24h)
- 🔴 Уровень риска (1-10)
- ⚠️ Причины рисков

## 🔐 Безопасность

- Используется официальный Telegram Bot API
- Никакие персональные данные не сохраняются
- Все данные о монетах из открытого CoinGecko API
- TOKEN хранится в переменных окружения
